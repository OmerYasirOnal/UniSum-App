# UniSum Backend API

Bu klasör, UniSum uygulamasının backend API servislerini içerir.

## Kurulum

```bash
npm install
```

## Geliştirme Ortamı

```bash
npm run dev
```

## Üretime Alma

```bash
npm start
```

## Çevre Değişkenleri

`.env.example` dosyasını `.env` olarak kopyalayıp içindeki değerleri güncelleyin.
